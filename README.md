# JokerNews 입니다.

## 프로젝트 설명
```
가짜뉴스를 올리는 커뮤니티 형식을 생각하고 제작하고 있습니다.
```

## 참조
```
./src/assets/config/config.dev.json 을 참조하세요
실제 서비스 에서는 ./src/assets/config/config.json을 사용합니다.
host.protocal = "http 또는 https입니다."
host.domain = "백엔드 주소입니다. 프로토콜을 제외하고 입력해주세요"
host.port = "포트가 없을 경우 제거 하시면 됩니다."
```

## Project Start
```
npm install
npm run serve
```
